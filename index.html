This repository contains the code. This repository is not maintained anymore and is just for reference purpose.

Overview

CNNs have been extensively used for image classification tasks, but their extension to videos poses a number of challenges. In this work, we try to recognize the actions based on the idea of converting the motion feature (Optical Flow) of videos into an image visualization and applying the CNN for action recognition on those images. As only a single motion feature is used for recognition, against the general notion of automatic extraction of spatio-temporal features using convolution operations (as may be done by applying 3D CNNs), the accuracy is expected to be low.

We plan to extend this work further by improving the results. Please refer the newer version of this repository for the complete code and link to the paper.
